Title                   : Hell is War
Filename                : hellis.pk3
Release date            : ?
Author                  : Jon Landis
Email Address           : jjjhhhlll@gmail.com

Description             : Hell is War is an episode (8 maps + 1 secret map)
                          for uzdoom.
                          
                          Each level uses the texture theme from the
                          corresponding level from doom2.  I'm only using
                          textures that appear in the doom2 level or custom
                          textures that only use patches that appear in the
                          doom2 level.
                          
                          All difficulty levels are implemented and all
                          difficulties have the same monsters.  Easier
                          difficulties have more generous items.
                          
                          Hell is War plays in the traditional doom way
                          (doom2.wad), without freelook, jumping or
                          crouching.  It uses uzdoom features such as 3d
                          floors, scripting and portals.  The hardware
                          renderer is recommended.
                          
                          All music is midi composed by me for fluidsynth
                          with the soundfont bundled with uzdoom. Special
                          thanks to @scwiba for getting me behind a sequencer
                          again.  (I forgot how fun it can be.)
                          
                          Maps:
                          
                          e1m1: Abdomen Strike (textures from "Entryway")
                          (music: "Berzerk Buzz") -- ready for playtesting
                          
                          e1m2: Waterboard (textures from "Underhalls")
                          (music: "Mainline")
                          
                          e1m3: Black Site (textures from "The Gantlet")
                          (music: "Suspicious Key")
                           
                          e1m4: Stress Position (textures from "The Focus") 
                          (music: "Empty Shell")
                           
                          e1m5: Rendition (textures from "The Waste Tunnels")
                           (music: "Sleet")
                          
                          e1m6: Collateral Damage (textures from "The
                          Crusher") (music: "Stuck in the Nukage") -- ready
                          for playtesting
                          
                          e1m7: Protective Custody (textures from "Dead
                          Simple") (music: "Megaton")
                          
                          e1m8: Shock and Awe (textures from "Tricks and
                          Traps") (music: "Bruxism")
                          
                          e1m9: Scanners (textures from all other maps in the
                          episode) (music: "Cratepusher")
                          
                          titlemap: (music: "Megaton")
                          intermission: (music: "Don't call")
                          exit: (music: "Circular Circus")
